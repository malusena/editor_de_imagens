Instruções para compilar
É recomendado que seja executado em ambiente Linux. Para compilar, basta digitar "make" no terminal, no diretorio ../maria_carvalho/. Ele vai executar o comando gcc main.c estrutura.c imagens_ppm.c -o main e vai gerar um arquivo executável chamado main. Após isso, basta executar ./main no terminal.

Instruções para editar as imagens
Para abrir o menu e ter instruções de como editar as imagens, basta executar o ./main no terminal. No menu há todas as informações necessárias para o usuário entender como editar as imagens e as funcionalidades de edição disponíveis.

Instruções para editar as imagens
Digite ./main suaimagem.ppm funcionalidade parametro suaimagemsaida.ppm
